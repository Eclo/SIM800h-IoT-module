Welcome

Welcome to the SIM800H .NETMF driver.

Getting Started

The best way to get started in using the driver is to go through the example projects that we provide with usage examples written in C#. Check the GitHub repository for the example projects. There example projects targeting .NETMF 4.3 and 4.4. The following example projects are available:

  • Basic initialization

  • Text message (SMS) send/receive

  • Text message (SMS) management

  • NTP client

  • HTTP client with GET (weather data)

  • HTTP client with POST (send data to ThingSpeak)

  • Get Time and Location

  • Power Mode example

  • MMS message send

Azure IoT Hub

Azure IoT Hub provides a great infra-structure for IoT use scenarios. The Azure IoT client SDK includes a version targeting .NETMF. We provide a special build of the SDK specific for the SIM800H module. Check the usage examples written in C# for the following topics:

  • HTTP client
